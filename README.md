# E-commerce

This a simple e-commerce web app built in JavaScript. This project fetches product data from the FakeStore API and allows users to browse products, view details and add items to their shopping cart. It is created as a beginner-friendly practice project.

## Project Overview

It displays a list of products, lets users search and filter items and provides a dedicated product details page. Users can add and remove products from the cart and the total price updates automatically.

## Features

- Fetch products from FakeStore API

- Product listing with images, price, category, and title

- Search and category filter

- View full product details

- Add to cart / Remove from cart

- Prevent duplicate cart items

- Cart total price calculation

- Simple and clean UI

## Technologies Used

- HTML

- JavaScript

- CSS

- FakeStore API
